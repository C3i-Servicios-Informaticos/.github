# C3i Servicios Informáticos

Empresa de servicios informáticos con sede en Murcia. Trabajamos con empresas de la Región de Murcia desde hace más de 15 años como partners tecnológicos.

En esta organización mantenemos el software que desarrollamos a medida para nuestros clientes y para nuestra propia operativa interna.

## 🔧 Áreas de trabajo

- Mantenimiento informático para empresas
- Desarrollo de software a medida (CRM, ERP, webs corporativas)
- Virtualización y soluciones de teletrabajo
- Ciberseguridad
- Digitalización de procesos y facturación electrónica
- Agentes digitalizadores oficiales del programa Kit Digital

## 💻 Tecnologías

Frontend: TypeScript, React, Next.js, Vue, Tailwind CSS
Backend: Node.js, NestJS, Laravel, PHP, Prisma
Datos e infraestructura: PostgreSQL, Redis, BullMQ, Docker
Plataformas: Odoo
